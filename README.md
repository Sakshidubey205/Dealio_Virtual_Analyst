#  DEALIO – Virtual Data Analyst

##  Project Overview

DEALIO is a data-driven analytics and recommendation system built to analyze smartphones and earphones datasets.
It provides insights, predictions, and personalized recommendations using machine learning and interactive dashboards.

---

##  Key Features

* **Smartphone Analysis Dashboard**
* **Earphones Analysis Dashboard**
* **Recommendation System**
* **Rating Prediction (ML Model)**
* **Price Prediction (ML Model)**
* **EDA Reports & Data Visualization**

---

##  Technologies Used

* Python 
* Pandas & NumPy
* Scikit-learn
* Matplotlib & Seaborn
* Streamlit (for web app interface)

##  How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Sakshidubey205/Dealio_Virtual_Analyst.git
```

2. Navigate to project folder:

```
cd dealio_major_project_final_folder
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run home.py
```
